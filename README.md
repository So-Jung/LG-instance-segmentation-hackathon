# LG 입자 형태 분석 대회

## 대회 개요 및 task
데이터셋을 활용해서 유체상의 떠다니는 유동입자의 윤곽을 검출하는 것.
task: instance segmentation 


## Directory

- EDA.ipynb
- dataset : image + annotation file

  - annotation file : 각 입자의 윤곽이 레이블링
